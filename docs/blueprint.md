# **App Name**: School Canteen Inventory

## Core Features:

- User Authentication: Secure login for authorized personnel using credentials.
- Product Registration: Register new products, including name, category, quantity, and expiration date.
- Product Search and Filtering: Quickly find items by name or category.
- Inventory Visualization: Graphical representation of inventory levels.
- Low Stock Alerts: Automated alerts when a product is running low. A tool is used to determine when to raise alerts.

## Style Guidelines:

- Primary color: Deep sky blue (#00BFFF) to reflect the trustworthiness and integrity associated with the educational sector. 
- Background color: Light grayish-blue (#E6F0FF), ensuring a clean and professional canvas.
- Accent color: Steel blue (#4682B4) for interactive elements like buttons, providing a clear visual cue.
- Font pairing: 'Inter' (sans-serif) for both body text and headers, creating a clean, modern and highly legible user experience.
- Simple, clear, and professional icons representing different product categories.
- Clean, organized, and intuitive layout for easy navigation.
- Subtle transitions and feedback animations to enhance user experience.
